
A responsive marketing landing page built with Vite + React + SCSS.

Live demo: https://landing-tau-flame.vercel.app  
Stack: Vite, React, SCSS, ES Modules

## Getting Started
~~~bash
npm install
npm run dev
~~~

## Build
~~~bash
npm run build
npm run preview
~~~
